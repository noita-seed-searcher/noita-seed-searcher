A list of the notable greater chests we found while scanning the first 40M seeds (coalmine + excavationsite, **main** world only, no parallel worlds). The data come from the `gsc_0-10m / 10-20m / 20-30m / 30-40m.out` dumps.

## Midas + HP-regen potion

These are all the **24** greater chests that hold a `midas` potion **and** a hp_regen potion at the same time. They are all the *unstable* hp_regen (`magic_liquid_hp_regeneration_unstable`), we never saw one with the rare stable hp_regen in 126,680 chests. Each block is the full content of the chest, copied as-is from the dump.
